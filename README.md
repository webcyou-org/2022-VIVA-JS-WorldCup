## How to Play

| Player | key                |
| :----- | :----------------- |
| ↑      | ArrowUp or KeyW    |
| →      | ArrowRight or KeyD |
| ↓      | ArrowDown or KeyS  |
| ←      | ArrowLeft or KeyA  |
| shoot  | KeyX               |
